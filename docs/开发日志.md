## 市场温度计算
4 项核心指标全部来自真实数据：
- **最高连板高度**: 涨停池 `lbc` 字段取 max
- **晋级率**: 昨涨停码 ∩ 今涨停码 / 昨涨停码
- **炸板率**: 炸板池 / (涨停池 + 炸板池)
- **昨涨停均收益**: 昨日涨停股的今日涨跌幅均值

阈值判断 → 四态输出：ICE (冰点) / WARM (回暖) / HIGH (高潮) / RETREAT (退潮)
