# 股票AI预测系统功能模块文档
## 一、首页：智能搜索
**业务流程**
1. 用户输入任意文本内容；
2. 经由LLM智能分类，结果分为三类：**个股 / 行业 / 未知**；
3. 分类为个股时，LLM 同时返回股票代码（代码缺失则按 DB → 东方财富 API 兜底查找）。

### 1.1 分类为【个股】链路
`个股 → 个股分析 → AI预测 → 按时间段预测 → 预测跟踪`
**预测跟踪内容：**
- 校验本次预测结果准确度；
- 梳理行情偏离预测的具体原因；
- 判断是否需要修正原有预测；
- 记录调整原因、历次调整全台账。

### 1.2 分类为【行业】链路
`行业 → 行业分析 → AI预测 → 短期(3个月)+长期(1年及以上)走势预测 → 预测跟踪`
**预测跟踪内容：**
- 核查行业是否发生影响原有预测的关键事件（行业重磅利好政策、黑天鹅突发事件等）；
- 判断是否需要调整行业预测结论；
- 记录调整原因、历次调整全台账。

### 1.3 分类为【未知】
无法归类为个股/行业，终止后续预测流程。

## 二、AI预测界面
### 2.1 数据展示分类
1. **股票预测记录**
点击个股名称：弹窗/跳转展示该标的全时段预测数据、跟踪记录、历史调整记录；
2. **行业预测记录**
点击行业名称：弹窗/跳转展示该行业预测数据、跟踪记录、历史调整记录。

### 2.2 操作与存储限制
- 列表项统一配备**删除按钮**，可手动删除单条预测记录；
- 存储上限规则：
  - 个股：最多保存 **10只** 股票预测记录
  - 行业：最多保存 **3个** 行业预测记录
- 超限逻辑：新增预测时超出上限，弹窗提示「预测数量已达上限，请删除已有记录后新增」；用户可删除存量记录，释放名额后新建预测。

## 三、持仓诊断界面
1. 导入用户持仓个股数据，系统自动对持仓标的逐一诊断；
2. 输出内容：个股后续行情预判、实操买卖操作建议；
3. 配套功能：**持仓复盘**
   - 自动生成复盘台账；
   - 核对实际走势与前期预测匹配度；
   - 结合复盘结果，判定原有操作/预测方案是否需要优化调整。

## 四、提示词配置界面
### 字段清单（单条提示词存储字段）
| 字段 | 说明 |
| ---- | ---- |
| 提示词名称 | 自定义命名 |
| 描述/适用场景 | 该提示词使用范围、功能说明 |
| 激活状态 | 启用/停用 |
| 更新时间 | 最后一次编辑保存时间 |

### 基础操作
支持**新增、删除、修改、查询（增删改查）**全功能。
