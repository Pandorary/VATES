首页：智能搜索
输入任意文本 → LLM 分类为个股/行业/未知  
个股->AI预测（按时间段：下一交易日/一周/1个月/3个月）->预测跟踪：判断预测是否准确，偏离原因，是否需要调整预测，调整原因，跟踪调整记录
行业->AI预测（短期1-3月 + 长期6月+）->预测跟踪：分析行业是否出现重大影响预测结果事件，比如重大利好政策，或者黑天鹅事件，是否需要调整预测，调整原因，跟踪调整记录

AI预测界面
股票预测记录，点击股票名称，展示不同时间段的预测结果和跟踪记录以及调整记录
行业预测记录，点击行业名称，展示行业预测结果跟踪记录以及调整记录
提供删除按钮
最多只能保存10只股票和3个行业的预测记录，超过提示预测超过限制。可以删除原来的预测个股和行业，以便预测新的股票

持仓诊断界面
根据持仓股，对持仓股进行诊断，预测未来的走向和操作建议
同样提供持仓股复盘，并且记录复盘记录，以及是否符合预测，是否需要调整

提示词界面
主要是保存提示词名称，描述/适用场景，状态是否激活，更新时间，以及增删改查
